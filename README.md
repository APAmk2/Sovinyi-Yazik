# Совиный язык
Совиный язык, шифровка и дешифровка

Есть два варианта:
* Консольный
* ГУИшный

У них ещё два вида: шифратор и дешифратор

* script.py/GUI_try1.py - шифруют
* DEscript.py.GUI_try2.py - ДЕшифруют

* PySimpleGUI (Только для ГУИ вариантов)
